# API Documentation

Base URL (local): `http://127.0.0.1:8000`
Interactive docs (auto-generated by FastAPI): `GET /docs` (Swagger) · `GET /redoc`

All endpoints return JSON. CORS is open so the dashboard can call them from the browser.

---

## `GET /health`

Liveness check.

**Response 200**
```json
{ "status": "ok" }
```

---

## `POST /rank`

Rank candidates for a job description and return the ranked shortlist as JSON.

**Request** — `multipart/form-data`

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `jd_text` | string | yes | Raw job description text |
| `top_k` | int | no (default 100) | Number of candidates to return |
| `candidates_file` | file | no | `candidates.jsonl` or `.jsonl.gz`. If omitted, uses `data/candidates.jsonl(.gz)` |

**Response 200** — `RankResponse`
```json
{
  "job_title": "Senior AI Engineer",
  "total_candidates": 100000,
  "results": [
    {
      "candidate_id": "CAND_0001930",
      "rank": 1,
      "score": 0.9871,
      "reasoning": "Senior Software Engineer (ML) with 6.9 yrs; matches on Embeddings; recruiter response rate 0.65."
    }
  ]
}
```

**Errors**
- `400` — no candidate file uploaded and no dataset found in `/data`
- `500` — pipeline failure (message included in `detail`)

**Example**
```bash
curl -X POST http://127.0.0.1:8000/rank \
  -F "jd_text=$(cat data/job_description.md)" \
  -F "top_k=100" \
  -F "candidates_file=@data/candidates.jsonl"
```

## `POST /rank/csv`

Same inputs as `/rank`, but writes the spec-compliant submission CSV to
`outputs/submission.csv` and returns its path. Use this to generate the actual
hackathon submission from the UI.

**Response 200**
```json
{ "rows": 100, "csv_path": ".../outputs/submission.csv" }
```

## Data contracts (`app/schemas.py`)

- **ParsedJD** — `required_skills[]`, `preferred_skills[]`, `experience_required_min/max`, `role_type`, `notes`
- **CandidateFeatures** — `candidate_id`, `name`, `current_title`, `years_of_experience`, `skills[]`, `summary`, behavioral signals, `embedding[]`, `raw`
- **ScoredCandidate** — `candidate_id`, `score`, `skill_match`, `experience_relevance`, `growth_index`, `behavioral_score`
- **RankedCandidate** — `candidate_id`, `rank`, `score`, `reasoning`

CSV submission columns (fixed order): `candidate_id,rank,score,reasoning`.
